### Integración continua
Provee las herramientas para permitir al desarrollador cumplir con todo el flujo de desarrollo.

Lo más común en un flujo de Integración continua son los siguientes pasos numerados:  

1. Gestión de Librerias o Instalaciones necesarias para que el proyecto funcione localmente.
2. Desarrollador genera el código.
3. Gestión de versionado.
4. Pruebas, En java puede ser: Jenkins (Para test), SonarQube para calidad del software.
5. Despliegue.
